# SQL Data Cleaning & Exploration

## 📌 Project Overview
This project focuses on cleaning, organizing, and exploring a dataset using SQL and Python. The workflow includes data ingestion, preprocessing, schema creation, SQL-based analysis, and documentation to ensure the dataset is ready for further analytics and reporting.

---

## 🎯 Objectives
- Import the raw dataset into a database.
- Clean and preprocess the data.
- Design an SQL database schema.
- Perform SQL-based data exploration.
- Validate data quality and consistency.
- Document the dataset and project workflow.

## ▶️ How to Run

1. Clone the repository.
2. Install the required dependencies.

```bash

```

3. Open the Jupyter notebook.

```bash
jupyter notebook notebooks/data_cleaning.ipynb
```

4. Execute all cells from top to bottom.
